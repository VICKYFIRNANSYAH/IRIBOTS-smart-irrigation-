🌱 IRIBOTS Smart Irrigation System

Sistem penyiraman tanaman otomatis berbasis IoT yang dikembangkan oleh tim IRIBOTS menggunakan Arduino Uno dan ESP8266. Sistem ini mampu memonitor kelembapan tanah dan melakukan penyiraman secara otomatis maupun manual melalui aplikasi.

Project ini memiliki beberapa mode operasi seperti Offline, Online menggunakan Blynk, serta komunikasi Bluetooth.

🚀 Fitur Utama

✅ Monitoring kelembapan tanah secara real-time
✅ Penyiraman tanaman otomatis berdasarkan sensor
✅ Kontrol manual melalui aplikasi IoT
✅ Mode online (IoT) dan offline
✅ Komunikasi Bluetooth RX-TX
✅ Integrasi dengan ESP8266 WiFi Module

🔧 Hardware yang Digunakan
Arduino UNO
ESP8266 WiFi Module
Soil Moisture Sensor
Relay Module
Water Pump
Bluetooth Module (HC-05 / HC-06)
Power Supply
